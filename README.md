**<h1>Primeira Challenge Fiap</h1>**

  <h2>Criação de um Fintech </h2>  
  
  <p>Desenvolvimento da primeira challenge, seguindo os aprendizados e conhecimentos do 1º e 2º semestre da faculdade de Análise e Desenvolvimento de Sistemas na Fiap.</p>
  
  <p>**Madu Finance** foi o nome que batizei, um sistema de gestão inteligente de fluxo de caixa e crédito. O objetivo é criar uma fintech que ajude o usuário a organizar suas finanças pessoais de forma automatizada e inteligente, eliminando a necessidade de lançamentos manuais e oferecendo alertas preditivos com base nos dados financeiros.</p>
